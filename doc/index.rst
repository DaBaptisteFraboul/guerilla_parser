guerilla_parser
===============

Python Guerilla file parser is a python module aimed to parse and retrieve
`Guerilla render`_ files content easily. It supports .gproject, .glayer, and
other files written by Guerilla.

.. _Guerilla render: http://guerillarender.com/

This module does not require any Python package beyond the standard ones.

If you don't know this module, the first thing to do is go the :doc:`example`
section to see various use cases.
